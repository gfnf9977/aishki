✨ Frontend та Дизайн-ресурси

Інструменти, анімації та джерела натхнення для розробника.

Анімації та CSS-ефекти

Ouidevcss

Колекція чистих CSS-скриптів: лоадери, бордери, градієнти та hover-ефекти.

Neat

Генератор анімованих 3D-градієнтів для фону сайту. Експорт у CSS/JSON.

CSS Loaders

Понад 600 лоадерів на чистому CSS. Копіювання коду в один клік.

Animista

Конструктор базових CSS анімацій з налаштуванням easing та delay.

Іконки та Графіка

SVG Repo

500,000+ безкоштовних SVG-векторів та іконок з відкритою ліцензією.

LottieFiles

Найбільша бібліотека легких анімованих іконок та ілюстрацій (JSON).

CSS.gg

700+ іконок у форматі чистого CSS. Мінімалістичні та легкі для UI.

Розробка та Дані

Faker API

Генерація мок-даних (користувачі, продукти, адреси) для тестування фронтенду.

CSSButtons

Колекція стильних анімованих кнопок. Частина спільноти Uiverse.

Dark Design

Land-book

Величезна добірка лендінгів, відсортована за типом та кольором.

Shadertoy
